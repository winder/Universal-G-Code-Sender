Getting started

Welcome to Universal Gcode Sender - If this is your first time using UGS thanks for giving it a try. This window is here to help you get started, once you are comfortable with the basic sending functionality please explore the other tabs to learn about the other features that this software has to offer.

Opening a file

Open your gcode file from the file menu. Once the file is loaded, there are a number of things you can do with it: inspect the tool paths from the Visualizer, edit them using the built in editor, see the number of rows in the program in the status bar. UGS is a great way to preview your tool paths even if you are not ready to run the program yet!

Connecting to a controller

When you are ready to connect to your machine, the serial connection can be configured in the toolbar at the top of the program. Choose your firmware from the combo box and configure the serial options accordingly. If you have not yet configured your controller you may be interested in the Setup wizard found in the Machine menu to help you set it up.

Once connected you can use UGS to control your machine. Some good places to start are the Jog Controller, manually sending commands in the console, or working with the coordinate system in the DRO. You can find more information about features and how to use them here.

Running your machine

Once you have loaded a file and established a connection to your controller, the program may be run. These icons in the toolbar can be used to stream, pause, and stopping the program.
